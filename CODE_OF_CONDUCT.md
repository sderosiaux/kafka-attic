# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. All contributors, maintainers, and community participants are expected to abide by it.

The full text — including the pledge, standards, scope, and enforcement responsibilities — is available at the link above. Please read it before participating.

## Reporting

Report unacceptable behaviour, harassment, or any concern about a community interaction to:

**community@conduktor.io**

Reports are read by Conduktor staff acting as project stewards. Reporter identity is kept confidential to the extent permitted by the response process. Retaliation against reporters is itself a violation of this code.

## Enforcement

Enforcement follows the [Contributor Covenant Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines), which define a four-step ladder (Correction → Warning → Temporary Ban → Permanent Ban) calibrated to the severity and pattern of the violation. Maintainers will apply these guidelines in good faith.
