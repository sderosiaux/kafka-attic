Cleanup script

Permanent data deletion. Read this section fully. Kafka has no native dry-run for topic deletion. Every command below will permanently destroy data.

Inclusion rules

  • Verdict is LIKELY_UNUSED (score ≥ 90)
  • No MISSING_SIGNAL flag
  • No COMPACTED flag
  • No REMOTE_STORAGE flag (MSK Tiered Storage, Confluent Cloud infinite retention)
  • All five ATTIC sub-signals have evidence KNOWN or ESTIMATED (no UNKNOWN)
{{if .Cleanup.Included}} 
#!/usr/bin/env bash
# kafka-attic cleanup script
# Cluster: {{.Cluster.Name}} ({{.Cluster.Bootstrap}})
# Generated: {{.GeneratedAt}}
set -euo pipefail
BS="{{.Cluster.Bootstrap}}"
{{range .Cleanup.Included}}
# ---- {{.Name}} ----{{if .OwnerLine}}
{{.OwnerLine}}{{end}}
# preflight
kafka-topics    --bootstrap-server $BS --describe --topic {{.Name}}
kafka-consumer-groups --bootstrap-server $BS --describe --all-groups --topic {{.Name}}
# delete
kafka-topics    --bootstrap-server $BS --delete --topic {{.Name}}
{{end}}
{{else}}

No topics satisfy the inclusion rules in this scan.

{{end}}
